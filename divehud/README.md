# Dive HUD
Dive HUD es un proyecto para hacer un HUD que se integre a cualquier (?) máscara de buceo.

## Ideas iniciales
- El strap de la máscara es el primer candidato para la integración.
- Si el strap no fuera estable para ajustar el display con comodidad, tal vez un montaje universal para action cameras sea el mejor sustituto.
- En la etapa 1 indicaría sólo profundidad.
- En la etapa 2 indicaría el tiempo de inmersión. El timer se activaría cuando el sensor lea 1.1ATM
- En la etapa 3 podría implementar algoritmos de absorción de nitrogeno como Buhlmann.

## Investigación
- El sensor MS5837 30BA mide presión hasta 30ATM
- Tener en cuenta que la batería tiene que durar no menos de 4 horas (dos buceos + intervalo de superficie). Planearlo bajo consumo desde el principio.
- El display podría ser un FLCoS
